\exercise{1}{
    Да се напише програма, която въвежда име и фамилия (не по-дълги от 128 символа)
    на човек и ги извежда във формат: \outputline{\code{<Last name>, <First name>}}
}
\example{Jason Statham}{Statham, Jason}
\bigskip

\exercise{2}{
    Да се напише функция:
    \outputline{\code{void reverse(char* str)}}, \\
    която обръща низ на място, т.е. без използване на допълнителни масиви.
}
\bigskip

\exercise{3}{
    Да се напише функция:

    \code{void caesar(const char* text, int key, char* result)},

    която шифрира първия низ по \href{https://en.wikipedia.org/wiki/Caesar_cipher}{шифъра на Цезар}
    и записва резултата в \code{result}.
}
\bigskip

\exercise{4}{
    Да се напише функция, която приема низ и връща дължината на най-дългия му непразен
    подниз, съставен от само един уникален символ.
}
\begin{multicols}{2}
    \example{inbetw\underline{ee}n}{2}
    \columnbreak
    \example{\underline{p}}{1}
\end{multicols}
\newpage

\exercise{5}{
    Да се напише функция:

    \code{void vigenere(const char* text, char* key, char* result)}.

    Функцията трябва да шифрира първия низ по
    \href{https://en.wikipedia.org/wiki/Vigen\%C3\%A8re_cipher}{шифъра на Виженер},
    използвайки втория низ като ключ, и да запише резултата в третия.
}
\bigskip

\exercise{6}{
    Да се напише функция, която прави всички букви в низ малки.
}
\bigskip

\exercise{7}{
    Да се напише функция, която сравнява два низа лексикографски
    (без да се използва \code{strcmp()})
}
\bigskip

\exercise{8}{
    Да се напише функция, която обръща всички думи в низ.
}
\bigskip

\exercise{9}{
    Да се напише функция, която изтрива всички срещания на даден низ \code{what}
    в даден низ \code{where}.

    Функцията да връща указател към позицията на първия изтрит подниз в \code{where} или
    \code{nullptr}, ако не е изтрит нито един подниз.
}
\example{
    \outputline{where: "aa bb aac ac"}
    \outputline{what:\hphantom{e} "aa"}
}{
    \outputline{where: "** bb **c ac"}
}
\bigskip

\exercise{10}{
    Да се напише програма, която въвежда низ с дължина до 256 символа и извежда следните
    статистики:
    \begin{itemize}
        \item{броят на думите;}
        \item{средната дължина на думите;}
        \item{най-дългата дума;}
    \end{itemize}
}
\bigskip

\exercise{11}{
    Да се напише функция, която приема низ, съдържащ само символите
    \code{(} и \code{)}, и индекс на отваряща скоба, и връща индекса
    на съответната ѝ затваряща скоба.
}
\newpage

\exercise{12}{
    Да се напише функция \outputline{\code{trim(char* str)}},

    която премахва \textit{whitespace}
    символи (празни места, табулации, нов ред) от началото и края на низ.
}
\bigskip

\exercise{13}{
    Да се напише функция \outputline{\code{concat(const char* str1, const char* str2)}},

    която връща указател към нов низ, образуван от долепянето на \code{str1} и \code{str2}.
}
\bigskip

\exercise{14}{
    Да се напише функция \outputline{\code{anagram(const char* str1, const char* str2)}},

    която връща дали два низа са анаграми или не. Низ е \textbf{анаграма} на друг низ,
    ако е пермутация на буквите му.
}
